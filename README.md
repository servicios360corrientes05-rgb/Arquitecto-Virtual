# 🏗️ Arquitecto Virtual & Ecosistema de Automatización

Este proyecto es el núcleo de **Arquitecto Virtual**, integrado en el entorno de **Google Antigravity**. Combina desarrollo moderno con Next.js, automatización de ventas para **Shopify** y gestión inteligente de datos.

## 🌟 Visión del Proyecto
Desarrollar una herramienta de alta eficiencia para profesionales de la construcción y bienes raíces, con una estética **moderna y futurista** (inspirada en paletas de negros profundos y rojos volcánicos).

## 🚀 Módulos y Capacidades

* **Asistente Virtual (BuilderBot):** Lógica de chat personalizada para atención al cliente.
* **Integración Shopify:** Scripts avanzados (`convert_tiendanube_shopify.py`, etc.) para migración y actualización de precios.
* **Gestión de Datos:** Conexión con Google Sheets ("Información de Clientes") para capturar leads en tiempo real.
* **Análisis de Potencial:** Procesamiento de normativas municipales y archivos PDF de planeamiento.

## 🛠️ Stack Tecnológico
* **Core:** Next.js (App Router)
* **Entorno de Ejecución:** Google Antigravity
* **Automatización:** GitHub CLI (gh), Python, Node.js
* **Pagos:** Configuración en curso con **Mercado Pago**.
